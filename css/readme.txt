Description : 
This is a website for a coffee shop named Creamy. In this website we can see what types of coffes are available in the store
and images for all types of coffees are attached in gallery. Social media info is given and store locations and timings are 
also mentioned. There is a form attached to contact the team for any details required. 

There are a total of 3 pages
1. Home pages
2. Gallery 
3. Contact

Components used in this website are 

Mixins
Variables
Nesting
form
footer
container
Used sass features like @include and @extend 
Placeholder selectors
favicons
